üöÄ Getting Started

Prerequisites

  1. Python 3.8+: Download Python
  2. Discord Bot Token: Create a bot on the Discord Developer Portal.
  3. Supabase Project: Create a free project on the Supabase Dashboard.
  4. Google Gemini API Key: Get a key from Google AI Studio.

Installation

  1. Clone or Download this Project: Get all the files and place them in a folder.
  2. Run the SQL Scripts: In your Supabase project dashboard, go to the SQL Editor. Find the .sql files in the project folder and run their contents one by one. This will create the necessary database tables.
  3. Install Dependencies: Open a terminal in the project folder and run: 
    pip install -r requirements.txt
  4. Configure Environment Variables: Create a file named .env in the main project folder. Open it and add your credentials. You can find your Supabase URL and Key in your Supabase project's API settings. 
    # .env file
DISCORD_TOKEN="YOUR_DISCORD_BOT_TOKEN_HERE"
    
GEMINI_API_KEY="YOUR_GOOGLE_GEMINI_API_KEY_HERE"
    
SUPABASE_URL="YOUR_SUPABASE_PROJECT_URL_HERE"
SUPABASE_KEY="YOUR_SUPABASE_SERVICE_ROLE_KEY_HERE"
  5. Run the Bot: Start the bot with the following command: 
    python main.py
  6. Invite the Bot: In your Discord Developer Portal, go to the "OAuth2" > "URL Generator" tab. Select the bot and applications.commands scopes. Then, under "Bot Permissions," grant it Administrator permissions (this is the easiest way to ensure all features work). Use the generated URL to invite it to your server.

ü§ñ Command List

HatchMate uses Slash (/) Commands for all features.

Moderation

  • /purge - Deletes a specified number of messages.
  • /kick [reason] - Kicks a member.
  • /ban [reason] - Bans a member.
  • /automod addword - Adds a word to the blocklist.
  • /automod removeword - Removes a word from the blocklist.
  • /automod listwords - Lists all blocked words.

Community & Economy

  • /level [@member] - Checks a user's level.
  • /balance [@member] - Checks a user's coin balance.
  • /daily - Claims your daily coin reward.
  • /leaderboard - Shows the server's richest users.
  • /coinflip - Bet on a 50/50 coin flip.
  • /slots - Play the emoji slot machine!

Event & Server Setup

  • /set_welcome - Sets the channel for welcome messages.
  • /roles add - Creates a new reaction role.
  • /setup rapidfire_event - Creates a new Rapid Fire event.
  • /setup rapidfire_answers - Opens a form to enter the answers for a Rapid Fire event.
  • /set_gemini_role - Sets the role required to use the /ask command.

Event Gameplay

  • /quiz create ... - Creates a standard, single-question quiz.
  • /quiz start - Starts a standard quiz.
  • /giveaway start ... - Starts a reaction-based giveaway.
  • /play rapidfire - Starts a pre-configured Rapid Fire event.
  • /play rapidfire_open - Opens the chat for the Rapid Fire answering phase.

Utility & AI

  • /serverinfo - Displays stats about the server.
  • /userinfo [@member] - Displays info about a user.
  • /ask - Ask a question to the Gemini AI model.
